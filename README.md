# interaction-vla：让交互模型拥有身体

在开源实时交互模型 JoyAI-VL-Interaction 上外挂连续动作专家，让 Reachy Mini 在全双工交互中（说话时、聆听时、沉默时）流畅输出拟人动作，并发布首个"交互中机器人动作"的开放基准。目标 CVPR 2027。

## 必读

- **[docs/2026-07-14-交互VLA设计文档_zh.md](docs/2026-07-14-交互VLA设计文档_zh.md)** —— 项目总设计（当前权威）。实施以此为唯一依据，与其它文档冲突时以它为准。
- `docs/plans/` —— 分阶段实施计划（任务书）。服务器端按当前阶段的计划执行。
- `docs/experiments/` —— 实验报告（服务器端每阶段末提交：结果+代码位置+踩坑，中文）。

## 协作模式

本机（设计/审查，Fable 5）⇄ GitHub ⇄ 服务器 8×昇腾910B（实施/实验，Opus 4.8）。

服务器端执行时遇到设计未覆盖的决策：小事自决并在实验报告中声明；大事（动作表示/数据 schema/评测协议的改动）停下写入报告"待决问题"一节，等本地裁决。

## 约定

- 所有产出（文档、注释、commit message、报告）用中文。
- pip 用清华镜像：`-i https://pypi.tuna.tsinghua.edu.cn/simple`。
- 目录规划（随实施逐步建立）：`data_gen/`（轨迹模板库与数据构造）、`expert/`（动作专家模型与训练）、`sim/`（MuJoCo 回放与压测）、`eval/`（指标与基准）。
