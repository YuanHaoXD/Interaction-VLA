## 2. 标注数据格式契约（datasets/example.json 同构）

```json
{"samples": [{
  "video_id": "XKyuguz1jHc", "video_duration_s": 22.4,
  "question": [{"content": "为什么她戴着独角兽角?", "time": 4.0}],
  "response": [[{"content": "只是今天想感受点魔法!", "time": 7.0, "action": "wiggle_antennas"}]]
}]}
```
- `response` 是二维列表（每个 question 对一组回复，一次可多句）；
- `action ∈ {nod, shake_head, tilt_head, wiggle_antennas, none}`，非法值降级为 `none`。

## 3. convert_sample 实现要点（对齐计划 Step 2）

- question 时刻 → `user_text` 事件 + `speech_span = (t, t+est_dur(content))`；
- 带 action 的 response → 模板 `events`（none 不进）+ `response` 时间线事件（含 `est_speech_dur_s`、`action_label`）；
- `est_dur`：**中文 5.5 字/秒、英文 14 字符/秒**，按 ASCII 占比线性插值，两侧统一（契约 C0）；
- `duration = min(最后事件+5s, 视频长, 120s 上限)`；
- 轨迹用 B2 的 `compose(duration, events, speech_spans, rng, limits)`（含 idle 微动+backchannel+限幅限速）；
- 输出经 `validate_episode` 校验（契约 C0）。

## 4. 批量驱动（Step 3）

- `run_pipeline` 用 `multiprocessing.Pool`（默认 min(cpu,16) 进程）；
- **增量**：episode 已存在且 `validate_episode` 通过 → 跳过（可断点续跑）；
- **失败隔离**：校验失败/异常的样本移入 `<out>_quarantine/` 并计数，不污染主目录；
- 数据产物**不入 git**（存服务器本地盘/`/cache`）。
