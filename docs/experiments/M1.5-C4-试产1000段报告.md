## 3. 发现的问题与修复

### 3.1 `backchannel_nods` 边界越界 ✅ 已修复
**问题**：当 speech_span 结束时间超出 duration_s 时，`traj[i:i+n]` 索引越界导致 `ValueError: operands could not be broadcast together with shapes (0,9) (X,9) (0,9)`。

**修复**：添加边界检查，确保 `e = min(e, duration_s)` 和 `i < T`。

### 3.2 多进程写入竞争 ⚠️ 待修复
**问题**：workers >1 时，多进程同时写入同一目录导致文件损坏（`EOFError: No data left in file` 或 `json.decoder.JSONDecodeError`）。

**影响**：阻塞并行生产，但单线程可正常运行。

**方案**：待决问题，需在 `_process_one` 或 `write_episode` 层使用临时文件+原子重命名，或加进程锁。
